## 주제 방향과 예시

이 블로그의 모든 포스트는 아래 다섯 가지 방향 중 하나에 해당해야 한다.

### 방향 1 — 청약·분양 실전 가이드

청약 자격, 가점 계산, 당첨 전략을 구체적으로 다룬다. 공식 청약홈보다 읽기 쉽게.

```
주제 예시:
- "청약 당첨 확률 높이는 방법 — 가점제 계산법 완전 정리"
- "청약통장 납입 인정 기준 — 월 10만원 vs 25만원 차이"
- "분양가 상한제 단지 청약 전략 — 당첨 후 실거주 의무 이해하기"
- "무순위 청약(줍줍) 조건과 방법 — 아무나 넣을 수 있는 건 아니다"
- "청약 1순위 조건 정리 — 지역·가입기간·납입횟수 한눈에"
```

### 방향 2 — 전세·월세 계약 실전

계약 전 확인 사항, 보증금 보호 방법, 분쟁 대처법을 다룬다.

```
주제 예시:
- "전세 계약 시 반드시 확인할 것 — 등기부등본, 근저당, 전입신고"
- "전세 사기 유형별 정리 — 깡통전세·이중계약 피하는 법"
- "임대차 3법 핵심 정리 — 계약갱신청구권 행사 방법과 한계"
- "전세보증보험 가입 방법 — HUG vs SGI vs HF 차이와 조건"
- "월세 세액공제 받는 법 — 연말정산에서 얼마나 돌아오나"
```

### 방향 3 — 임장·매매 실전

현장에서 체크해야 할 것, 매매 프로세스, 협상 포인트를 다룬다.

```
주제 예시:
- "아파트 임장 체크리스트 — 현장에서 반드시 봐야 할 것들"
- "매매 계약서 작성 전 확인 사항 — 특약 조항 꼭 넣어야 하는 것"
- "부동산 등기부등본 보는 법 — 갑구·을구 각각 뭘 확인하나"
- "아파트 관리비 내역서 보는 법 — 이 단지가 잘 관리되는지 알 수 있다"
- "신축 vs 구축 — 어느 쪽이 실거주자한테 유리한가"
```

### 방향 4 — 세금·절세 실전 계산

양도소득세, 취득세, 종부세 계산법을 실제 숫자로 보여준다.

```
주제 예시:
- "부동산 양도소득세 계산법 — 1주택 비과세 기준과 예외"
- "취득세 계산법 — 주택 수·가격별로 세율이 어떻게 달라지나"
- "종합부동산세 대상자 확인 방법 — 2026년 기준 공시가격 기준"
- "증여세 vs 양도세 — 부모 명의 부동산 물려받을 때 뭐가 유리한가"
- "다주택자 양도소득세 중과 — 어떤 상황에서 얼마나 더 내나"
```

### 방향 5 — 부동산 시장 분석·정책 해설

오늘 발표된 정책, 금리 변동, 지역 이슈를 실수요자 관점에서 풀어준다.

```
주제 예시:
- "전월세 전환율 계산법 — 세입자가 손해 안 보는 기준선"
- "주택담보대출 DSR 규제 — 실제로 얼마까지 빌릴 수 있나"
- "부동산 공시가격 2026 — 재산세·건보료에 어떤 영향을 주나"
- "토지거래허가구역 지정 — 규제 지역에서 매매할 때 달라지는 것"
- "재개발·재건축 투자 전 알아야 할 것 — 사업 단계별 리스크"
```

## 글쓰기 스타일

### 언어 수준

- 부동산 용어를 쓸 때는 괄호 안에 쉬운 설명 추가
  - "근저당(채권최고액을 담보로 설정한 담보물권)"
  - "DSR(총부채원리금상환비율 — 연 소득 대비 모든 대출 원리금 비율)"
- 숫자와 계산 예시 필수 — 추상적 설명 금지
- 길이: 1,200~2,500자 (계산 예시 포함 시 더 길어도 OK)

### 톤

- 실제로 집 알아본 사람의 관점: "이거 처음엔 몰랐는데", "계약서 쓰고 나서야 알았다"
- 솔직하게: 함정·주의사항을 먼저 알려주는 스타일
- 반말 직접체: "~한다", "~이다", "확인해야 한다"

### AI 감지 표현 금지 (이 블로그 추가)

```
- "부동산 시장이 급변하고 있습니다"

- "다양한 요소를 고려해야 합니다"
- "전문가와 상담하는 것이 좋습니다"
- "투자는 신중하게"
- "꼼꼼하게 확인하시기 바랍니다"
```
